# Desktop CRM

This project provides a simple desktop application that makes working with your CRM a little easier. 

## Supported CRMs

- Sugar CRM
- Suite CRM

## Current Features

- Call notifications
- Task list
- Simple task reschedule
- Task due notification
- Follow up reminders for cases, opportunities and leads
